----------------

Tool : ChatGPT
Date : 20-10-2025

Prompt:
--------

When running my Flask weather app, I sometimes get requests.exceptions.ConnectionError or empty JSON responses. How can I handle these gracefully in Python and display user-friendly messages?

AI Response (code excerpt):
----------------------------

try:
    data = requests.get(url, timeout=10)
    data.raise_for_status()
    weather = data.json()
except requests.exceptions.RequestException:
    weather = None
    flash("Weather service temporarily unavailable. Please try again later.", "warning")

Before:
-------
App crashed with traceback on connection failure.

After:
------
- Wrapped API calls with `try-except`
- Displayed messages with Flask `flash()`
- Added fallback HTML (“Service Unavailable”) in `templates/error.html`

Reflection:
------------
Improved stability and user experience under poor network conditions.
