Provide Python code to visualize weather temperature and humidity over 3 days using matplotlib. The chart should include labels, legends, and date on the x-axis.

AI Response (code excerpt):

import matplotlib.pyplot as plt

def plot_weather(days, temps, humidity):
    plt.plot(days, temps, label="Temperature (°C)")
    plt.plot(days, humidity, label="Humidity (%)")
    plt.xlabel("Day")
    plt.ylabel("Values")
    plt.title("3-Day Weather Forecast")
    plt.legend()
    plt.show()
